# 图标文件说明

由于无法直接生成PNG图片文件，请按照以下方法添加图标：

## 方法一：使用在线工具（推荐）

1. 访问 https://favicon.io/emoji-favicons/ 或 https://www.favicon-generator.org/
2. 选择一个颜色相关的emoji（如🎨）或上传图片
3. 生成16x16、48x48、128x128三个尺寸
4. 保存为 icon16.png、icon48.png、icon128.png
5. 放入此 icons 文件夹

## 方法二：使用Photoshop/画图工具

1. 创建新图像（16x16像素）
2. 填充任意颜色（建议蓝色系 #4CAF50）
3. 保存为 icon16.png
4. 重复创建48x48和128x128版本

## 方法三：临时测试（快速方案）

创建一个简单的文本文件作为占位符，插件仍可加载但图标显示为默认：

```powershell
# 在PowerShell中运行
cd E:\data\mojh\ai_item\chang_color\icons
# 下载一个简单的彩色方块图标
Invoke-WebRequest -Uri "https://via.placeholder.com/16/4CAF50/FFFFFF?text=C" -OutFile "icon16.png"
Invoke-WebRequest -Uri "https://via.placeholder.com/48/4CAF50/FFFFFF?text=Color" -OutFile "icon48.png"
Invoke-WebRequest -Uri "https://via.placeholder.com/128/4CAF50/FFFFFF?text=ChangeColor" -OutFile "icon128.png"
```

## 方法四：复制现有图标

如果你有其他扩展的图标文件，可以直接复制过来并重命名。

## 图标设计要求

- **格式**：PNG（支持透明背景）
- **尺寸**：16x16、48x48、128x128 像素
- **内容**：建议使用调色板、画笔等与颜色相关的图案
- **风格**：简洁明了，在小尺寸下仍清晰可辨

## 验证图标

添加图标后，在 Edge 浏览器中：
1. 访问 `edge://extensions/`
2. 查看插件卡片是否显示正确的图标
3. 如果显示默认拼图图标，说明图标文件有问题
