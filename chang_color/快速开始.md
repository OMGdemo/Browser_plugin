# 快速开始 - 3步安装插件

## ✅ 已完成
所有必要文件已生成，包括图标文件！

## 📦 安装步骤（3步完成）

### 第1步：打开Edge扩展页面
在Edge浏览器地址栏输入：
```
edge://extensions/
```

### 第2步：开启开发者模式
- 找到页面左侧的 **"开发人员模式"** 开关
- 点击开启（开关变为蓝色）

### 第3步：加载插件
1. 点击顶部出现的 **"加载解压缩的扩展"** 按钮
2. 在弹出的文件夹选择窗口中，选择：
   ```
   E:\data\mojh\ai_item\chang_color
   ```
3. 点击"选择文件夹"

## 🎉 完成！

插件已成功安装！你会看到：
- 插件名称：**页面背景变色器**
- 版本：1.0.0
- 工具栏出现插件图标（绿色方块）

## 🎨 使用方法

1. **打开任意网页**（如 www.baidu.com）
2. **点击工具栏的插件图标**
3. **在弹出窗口中**：
   - 点击颜色选择器选择喜欢的颜色
   - 或直接点击预设颜色
   - 点击"应用颜色"按钮
4. **页面背景立即改变！**

## 🔧 常用操作

- **更改颜色**：重新打开插件，选择新颜色，点击应用
- **恢复默认**：点击"恢复默认"按钮
- **跨标签同步**：在一个标签页修改颜色，所有标签页自动更新

## ⚠️ 注意事项

- 某些系统页面无法修改（如 edge://settings）
- 刷新页面后颜色会保持
- 关闭浏览器后重新打开，颜色设置仍然有效

## 🐛 遇到问题？

### 插件不显示？
- 确认已开启"开发人员模式"
- 检查是否选择了正确的文件夹（包含manifest.json的目录）

### 颜色不生效？
- 刷新当前网页
- 尝试其他网页（排除系统页面）

### 图标是默认的？
- 右键插件 → "管理扩展" → 禁用后重新启用
